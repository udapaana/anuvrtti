## concept - @term[kṛtya] — "To Be Done"

@term[kṛtya] (gerundive/future passive participle) expresses necessity, obligation, or fitness — "X should/must/ought to be done." Three @term[pratyaya] create @term[kṛtya]: @deva[तव्यत्], @deva[अनीयर्], and @deva[यत्] / @deva[ण्यत्].

**Core pattern:** root + suffix → adjective meaning "to be X-ed"

| Root        | @deva[तव्य]        | @deva[अनीय]      | @deva[य]       | Meaning             |
| ----------- | ------------------ | ---------------- | -------------- | ------------------- |
| @deva[कृ]   | @deva[कर्तव्यम्]   | @deva[करणीयम्]   | @deva[कार्यम्] | to be done          |
| @deva[पठ्]  | @deva[पठितव्यम्]   | @deva[पठनीयम्]   | —              | to be read          |
| @deva[गम्]  | @deva[गन्तव्यम्]   | @deva[गमनीयम्]   | —              | to be gone to       |
| @deva[दा]   | @deva[दातव्यम्]    | @deva[दानीयम्]   | @deva[देयम्]   | to be given         |
| @deva[भू]   | @deva[भवितव्यम्]   | @deva[भवनीयम्]   | —              | to be (must happen) |
| @deva[वच्]  | @deva[वक्तव्यम्]   | @deva[वचनीयम्]   | —              | to be said          |
| @deva[पा]   | @deva[पातव्यम्]    | @deva[पानीयम्]   | @deva[पेयम्]   | to be drunk         |
| @deva[दृश्] | @deva[द्रष्टव्यम्] | @deva[दर्शनीयम्] | —              | to be seen          |

All three convey the same meaning; @deva[तव्य] and @deva[अनीय] are the most common. The @deva[यत्] / @deva[ण्यत्] forms are more restricted but some are very frequent (@deva[कार्यम्], @deva[देयम्], @deva[पेयम्]).

Gerundives agree with the noun in gender, number, and case (they decline like @deva[अ]-stem adjectives): @deva[कर्तव्यम्] (n.), @deva[कर्तव्यः] (m.), @deva[कर्तव्या] (f.).

## @ref[3.1.96] - तव्यत्तव्यानीयरः

**Key Terms:** @deva[तव्य], @deva[अनीयर्]

The affixes @deva[तव्यत्], @deva[तव्य], and @deva[अनीयर्] come after verbal roots. The final @deva[त्] of @deva[तव्यत्] and the @deva[र्] of @deva[अनीयर्] are indicatory (for accent marking).

**Realized forms:**

- @deva[तव्यत्] → @deva[-तव्य]
- @deva[अनीयर्] → @deva[-अनीय]

Both are @deva[आर्धधातुक] (see @ref[3.4.114]), which means @deva[गुण] and @deva[इट्] rules apply.

**Formation examples:**

- @deva[कृ] + @deva[तव्य] = @deva[कर्तव्यम्] "to be done"
- @deva[कृ] + @deva[अनीय] = @deva[करणीयम्] "to be done"
- @deva[श्रु] + @deva[तव्य] = @deva[श्रोतव्यम्] "to be heard"
- @deva[श्रु] + @deva[अनीय] = @deva[श्रवणीयम्] "to be heard"

---

## @ref[3.1.97] - अचो यत्

**Key Terms:** @deva[यत्], @deva[अच्]

The affix @deva[यत्] comes after a root ending in a vowel. This creates a third set of gerundive forms:

- @deva[गा] + @deva[यत्] = @deva[गेयम्] "to be sung"
- @deva[पा] + @deva[यत्] = @deva[पेयम्] "to be drunk"
- @deva[जि] + @deva[यत्] = @deva[जेयम्] "to be conquered"
- @deva[भू] + @deva[यत्] = @deva[भव्यम्] "to be, destined"

@deva[यत्] forms are more literary and often lexicalized as independent words (e.g., @deva[पानीयम्] "drinking water" from @deva[पा] + @deva[अनीय]).

---

## @ref[3.4.114] - आर्धधातुकं शेषः

**Key Terms:** @deva[आर्धधातुक], @deva[तव्य]

Affixes other than @deva[तिङ्] and those marked @deva[शित्] are @deva[आर्धधातुक]. Since @deva[तव्य], @deva[अनीय], and @deva[यत्] fall in this category, the @deva[इट्] augment and @deva[गुण] substitution rules apply to them.

This classification is what drives the stem changes in gerundive formation.

---

## @ref[7.2.35] - आर्धधातुकस्येड् वलादेः

**Key Terms:** @deva[आर्धधातुक], @deva[इट्], @deva[तव्य]

An @deva[आर्धधातुक] affix beginning with a @deva[वल्] consonant gets @deva[इट्] (= @deva[इ]).

@deva[तव्य] begins with @deva[त्] (a @deva[वल्] consonant), so @deva[सेट्] roots take @deva[इट्]:

| Root type    | Root        | + @deva[इट्]? | + @deva[तव्य]      | Result        |
| ------------ | ----------- | ------------- | ------------------ | ------------- |
| @deva[सेट्]  | @deva[पठ्]  | yes           | @deva[पठितव्यम्]   | to be read    |
| @deva[सेट्]  | @deva[भू]   | yes           | @deva[भवितव्यम्]   | must be       |
| @deva[सेट्]  | @deva[गम्]  | yes           | @deva[गन्तव्यम्]   | to be gone to |
| @deva[अनिट्] | @deva[कृ]   | no            | @deva[कर्तव्यम्]   | to be done    |
| @deva[अनिट्] | @deva[वच्]  | no            | @deva[वक्तव्यम्]   | to be said    |
| @deva[अनिट्] | @deva[दृश्] | no            | @deva[द्रष्टव्यम्] | to be seen    |

@deva[अनीय] begins with a vowel, so @deva[इट्] does not apply (it only applies to @deva[वल्]-initial affixes).

---

## @ref[7.3.84] - सार्वधातुकार्धधातुकयोः

**Key Terms:** @deva[गुण], @deva[तव्य]

@deva[गुण] substitutes for the final @deva[इक्] of a stem before @deva[आर्धधातुक] affixes:

- @deva[भू] → @deva[भो] → @deva[भव] (by @deva[गुण] + @deva[अयादेश]) → @deva[भवितव्यम्]
- @deva[नी] → @deva[ने] → @deva[नेतव्यम्]
- @deva[कृ] → @deva[कर्] (by @deva[गुण]) → @deva[कर्तव्यम्]
- @deva[श्रु] → @deva[श्रो] → @deva[श्रोतव्यम्]

---

## @ref[6.4.65] - एचोऽयवायावः

**Key Terms:** @deva[एच्], @deva[यत्]

Before a vowel-initial affix, the diphthongs @deva[ए], @deva[ओ], @deva[ऐ], @deva[औ] undergo @deva[आयादेश] (semi-vowel substitution):

- @deva[गा] → @deva[गे] (by @deva[गुण]) → @deva[ग् + अय्] (by @deva[आयादेश]) → + @deva[य] = @deva[गेयम्]
- @deva[पा] → @deva[पे] → @deva[पेयम्]

This is how the @deva[यत्] gerundives of vowel-final roots are formed.

---

## @ref[6.4.66] - घुमास्थागापाजहातिसां हलि

**Key Terms:** @deva[घु], @deva[अनीय]

For @deva[घु] roots (@deva[दा], @deva[धा]) and @deva[मा], @deva[स्था], @deva[गा], @deva[पा], @deva[हा], @deva[सा] — @deva[ई] substitutes for the root vowel before consonant-initial @deva[कित्] or @deva[ङित्] affixes.

This produces:

- @deva[दा] + @deva[यत्] → @deva[देयम्] "to be given"
- @deva[धा] + @deva[यत्] → @deva[धेयम्] "to be placed"
- @deva[पा] + @deva[यत्] → @deva[पेयम्] "to be drunk"

---

## @ref[3.1.100] - गदमदचरयमश्चानुपसर्गे

**Key Terms:** @deva[ण्यत्], @deva[गद्]

After certain roots without @deva[उपसर्ग] (prefix), @deva[ण्यत्] (= @deva[य] with @deva[वृद्धि]) applies instead of @deva[यत्]:

- @deva[कृ] + @deva[ण्यत्] = @deva[कार्यम्] "to be done" (@deva[वृद्धि]: @deva[कृ] → @deva[कार्])
- @deva[मद्] + @deva[ण्यत्] = @deva[माद्यम्] → @deva[मद्यम्] "to be intoxicated (= wine)"

The @deva[ण्] of @deva[ण्यत्] triggers @deva[वृद्धि] of the first vowel. This is why @deva[कार्यम्] has @deva[आ] (not @deva[अ] as in @deva[कर्तव्यम्]).

## concept - Gerundives in Literature

Gerundives are extremely common in Sanskrit, especially in legal, philosophical, and practical texts:

**1. Expressing duty (@deva[कर्तव्यता]):**

- @deva[सत्यं वक्तव्यम्] "Truth should be spoken"
- @deva[धर्मः रक्षितव्यः] "Dharma must be protected"
- @deva[गुरुः सेवितव्यः] "The teacher should be served"

**2. Necessity / inevitability:**

- @deva[भवितव्यं तथा तत्] "That must happen so" (destiny)
- @deva[मर्तव्यम्] "One must die" (from @deva[मृ])

**3. Worth / fitness:**

- @deva[दर्शनीयः] "worth seeing, handsome"
- @deva[श्रवणीयम्] "worth hearing"
- @deva[पूजनीयः] "worthy of worship"

**4. Lexicalized nouns:**

- @deva[पानीयम्] (from @deva[पा] + @deva[अनीय]) = "water" (lit. "to be drunk")
- @deva[भोजनीयम्] = "food" (lit. "to be eaten")
- @deva[कार्यम्] (from @deva[कृ] + @deva[ण्यत्]) = "task, affair" (lit. "to be done")

**Common gerundives to memorize:**

| Form                                  | Root        | Meaning       |
| ------------------------------------- | ----------- | ------------- |
| @deva[कर्तव्यम्] / @deva[कार्यम्]     | @deva[कृ]   | to be done    |
| @deva[वक्तव्यम्]                      | @deva[वच्]  | to be said    |
| @deva[द्रष्टव्यम्] / @deva[दर्शनीयम्] | @deva[दृश्] | to be seen    |
| @deva[श्रोतव्यम्] / @deva[श्रवणीयम्]  | @deva[श्रु] | to be heard   |
| @deva[ज्ञातव्यम्]                     | @deva[ज्ञा] | to be known   |
| @deva[देयम्] / @deva[दातव्यम्]        | @deva[दा]   | to be given   |
| @deva[पेयम्] / @deva[पानीयम्]         | @deva[पा]   | to be drunk   |
| @deva[गन्तव्यम्] / @deva[गमनीयम्]     | @deva[गम्]  | to be gone to |
