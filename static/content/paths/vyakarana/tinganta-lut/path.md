## concept - @deva[लुट्] — Periphrastic Future

@deva[लुट्] is used for events **not happening today** (@term[anadyatana]) — typically a more remote or definite @term[bhaviṣyat] (future). Unlike @deva[लृट्] (which uses the @deva[स्य] @term[vikaraṇa]), @deva[लुट्] uses the agent noun @term[pratyaya] @deva[तासि] (= @deva[ता]) combined with forms of @deva[अस्] "to be."

**Core pattern:** root + (@deva[इ]) + @deva[ता] (+ auxiliary @deva[अस्])

**Paradigm of @deva[कृ] (do) — @deva[लुट्] @deva[परस्मैपद]:**

|     | Singular         | Dual             | Plural           |
| --- | ---------------- | ---------------- | ---------------- |
| 3rd | @deva[कर्ता]     | @deva[कर्तारौ]   | @deva[कर्तारः]   |
| 2nd | @deva[कर्तासि]   | @deva[कर्तास्थः] | @deva[कर्तास्थ]  |
| 1st | @deva[कर्तास्मि] | @deva[कर्तास्वः] | @deva[कर्तास्मः] |

**Paradigm of @deva[भू] (be) — @deva[लुट्]:**

|     | Singular         | Dual             | Plural           |
| --- | ---------------- | ---------------- | ---------------- |
| 3rd | @deva[भविता]     | @deva[भवितारौ]   | @deva[भवितारः]   |
| 2nd | @deva[भवितासि]   | @deva[भवितास्थः] | @deva[भवितास्थ]  |
| 1st | @deva[भवितास्मि] | @deva[भवितास्वः] | @deva[भवितास्मः] |

Note: The 3rd person forms are purely the agent noun (@deva[कर्ता], @deva[कर्तारौ], @deva[कर्तारः]). The 1st and 2nd person forms combine the agent noun with present tense of @deva[अस्] (@deva[अस्मि], @deva[असि], etc.).

**Comparison with @deva[लृट्]:**

| Root       | @deva[लृट्] (simple future) | @deva[लुट्] (periphrastic) |
| ---------- | --------------------------- | -------------------------- |
| @deva[कृ]  | @deva[करिष्यति]             | @deva[कर्ता]               |
| @deva[भू]  | @deva[भविष्यति]             | @deva[भविता]               |
| @deva[गम्] | @deva[गमिष्यति]             | @deva[गन्ता]               |
| @deva[दा]  | @deva[दास्यति]              | @deva[दाता]                |
| @deva[वच्] | @deva[वक्ष्यति]             | @deva[वक्ता]               |

## @ref[3.3.15] - अनद्यतने लुट्

**Key Terms:** @deva[अनद्यतन], @deva[लुट्]

The affix @deva[लुट्] comes after a verbal root in the sense of what will happen **not in the course of today** (@deva[अनद्यतन]).

This is the key distinction: @deva[लृट्] is the general future; @deva[लुट्] specifically marks a non-today future:

- @deva[श्वः कर्ता] "He will do (it) tomorrow"
- @deva[श्वो भोक्ता] "He will eat tomorrow"
- @deva[परश्वः गन्ता] "He will go day after tomorrow"

In practice, @deva[लृट्] is used far more commonly; @deva[लुट्] appears in Vedic literature and when emphasizing the certainty or definite nature of a future event.

## @ref[3.1.33] - स्यतासी लृलुटोः

**Key Terms:** @deva[तासि], @deva[लुट्]

When @deva[लुट्] follows, the suffix @deva[तासि] is added after the root (while @deva[लृट्] gets @deva[स्य]).

@deva[तासि] is realized as @deva[ता]: the @deva[स्] and @deva[इ] are indicatory. The @deva[इ] of @deva[तासि] indicates that root-final nasals are not dropped:

- @deva[मन्] + @deva[तासि] → @deva[मन्ता] "he will think" (nasal retained)
- @deva[कृ] + @deva[तासि] → @deva[कर्ता] "he will do"
- @deva[गम्] + @deva[तासि] → @deva[गन्ता] "he will go"
- @deva[दा] + @deva[तासि] → @deva[दाता] "he will give"

The resulting form (@deva[कर्ता], @deva[दाता], etc.) looks identical to the agent noun — and that is the design: the periphrastic future literally says "he is a doer" for "he will do."

## @ref[2.4.85] - लुटः प्रथमस्य डारौरसः

**Key Terms:** @deva[लुट्], @deva[प्रथम], @deva[डा]

@deva[डा], @deva[रौ], and @deva[रस्] replace the three 3rd-person endings of @deva[लुट्]:

| Original ending       | Replacement | Result         |
| --------------------- | ----------- | -------------- |
| @deva[तिप्] (3rd sg.) | @deva[डा]   | @deva[कर्ता]   |
| @deva[तस्] (3rd du.)  | @deva[रौ]   | @deva[कर्तारौ] |
| @deva[झि] (3rd pl.)   | @deva[रस्]  | @deva[कर्तारः] |

These replacements produce the agent-noun forms directly. The 3rd person forms do **not** use the @deva[अस्] auxiliary — they stand alone as agent nouns functioning as futures.

## @ref[3.4.78] - तिप्तस्झिसिप्थस्थमिब्वस्मस्तातांझथासाथांध्वमिड्वहिमहिङ्

**Key Terms:** @deva[तिप्], @deva[लुट्]

The standard @deva[तिङ्] endings substitute for @deva[ल]. For 1st and 2nd person @deva[लुट्], these endings are attached to the @deva[-ता] stem, producing a compound of agent noun + @deva[अस्]:

- 2nd sg.: @deva[कर्ता] + @deva[असि] → @deva[कर्तासि] "you will do"
- 1st sg.: @deva[कर्ता] + @deva[अस्मि] → @deva[कर्तास्मि] "I will do"
- 2nd pl.: @deva[कर्ता] + @deva[स्थ] → @deva[कर्तास्थ] "you all will do"

## @ref[7.2.10] - एकाच उपदेशेऽनुदात्तात्

**Key Terms:** @deva[इट्], @deva[अनुदात्त]

@deva[इट्] is not added before an affix following a monosyllabic @deva[अनुदात्त] root. So @deva[अनिट्] roots take @deva[तासि] directly:

| Type         | Root       | @deva[लुट्] form | Has @deva[इट्]? |
| ------------ | ---------- | ---------------- | --------------- |
| @deva[अनिट्] | @deva[कृ]  | @deva[कर्ता]     | No              |
| @deva[अनिट्] | @deva[नी]  | @deva[नेता]      | No              |
| @deva[अनिट्] | @deva[वच्] | @deva[वक्ता]     | No              |
| @deva[सेट्]  | @deva[भू]  | @deva[भविता]     | Yes             |
| @deva[सेट्]  | @deva[गम्] | @deva[गन्ता]     | Yes (special)   |
| @deva[सेट्]  | @deva[पठ्] | @deva[पठिता]     | Yes             |

## concept - @deva[लुट्] in Literature

The periphrastic future is relatively rare in Classical Sanskrit but meaningful when it appears:

**1. Definite future with time adverbs:**

- @deva[श्वः कर्ता] "He will (definitely) do it tomorrow"
- @deva[मासात् गन्ता] "He will go in a month"

**2. Agent-noun overlap:**
The @deva[लुट्] 3rd person forms are indistinguishable from agent nouns:

- @deva[कर्ता] = "doer" (agent noun) = "he will do" (@deva[लुट्])
- @deva[दाता] = "giver" = "he will give"
- @deva[हर्ता] = "taker" = "he will take"

Context determines which reading applies. In @deva[श्वः कर्ता], the time adverb @deva[श्वः] ("tomorrow") signals the future tense reading.

**Common @deva[लुट्] / agent noun forms:**

| Form           | Root        | As agent | As future  |
| -------------- | ----------- | -------- | ---------- |
| @deva[कर्ता]   | @deva[कृ]   | doer     | will do    |
| @deva[दाता]    | @deva[दा]   | giver    | will give  |
| @deva[भोक्ता]  | @deva[भुज्] | enjoyer  | will enjoy |
| @deva[वक्ता]   | @deva[वच्]  | speaker  | will speak |
| @deva[द्रष्टा] | @deva[दृश्] | seer     | will see   |
| @deva[श्रोता]  | @deva[श्रु] | listener | will hear  |
| @deva[नेता]    | @deva[नी]   | leader   | will lead  |
