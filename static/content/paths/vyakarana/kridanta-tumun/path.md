## concept - The Sanskrit Infinitive

The infinitive (@deva[तुमन्तम्]) is a verbal noun ending in @deva[-तुम्] that expresses purpose ("in order to do X") or serves as a complement to verbs like "to be able," "to wish," "to begin." It is indeclinable and always takes the accusative sense.

**Core pattern:** root + (@deva[इ]) + @deva[तुम्]

| Root        | Meaning | Infinitive       | Translation |
| ----------- | ------- | ---------------- | ----------- |
| @deva[कृ]   | do      | @deva[कर्तुम्]   | to do       |
| @deva[गम्]  | go      | @deva[गन्तुम्]   | to go       |
| @deva[पठ्]  | read    | @deva[पठितुम्]   | to read     |
| @deva[भू]   | be      | @deva[भवितुम्]   | to be       |
| @deva[नी]   | lead    | @deva[नेतुम्]    | to lead     |
| @deva[दा]   | give    | @deva[दातुम्]    | to give     |
| @deva[पच्]  | cook    | @deva[पक्तुम्]   | to cook     |
| @deva[वच्]  | speak   | @deva[वक्तुम्]   | to speak    |
| @deva[दृश्] | see     | @deva[द्रष्टुम्] | to see      |
| @deva[श्रु] | hear    | @deva[श्रोतुम्]  | to hear     |

Whether the linking vowel @deva[इ] (@deva[इट्]) appears depends on the root class (see @ref[7.2.35] and @ref[7.2.10] below).

## @ref[3.3.10] - तुमुन्ण्वुलौ क्रियायां क्रियार्थायाम्

**Key Terms:** @deva[तुमुन्], @deva[क्रिया], @deva[क्रियार्था]

The affixes @deva[तुमुन्] and @deva[ण्वुल्] are placed after a verbal root when the word in construction is another verb denoting an action performed **for the sake of** that future action.

This establishes the primary use: **purpose clauses**.

**Examples:**

- @deva[भोक्तुं व्रजति] "he goes **to eat**"
- @deva[पठितुं गच्छति] "he goes **to study**"
- @deva[द्रष्टुं इच्छति] "he wishes **to see**"
- @deva[कर्तुं शक्नोति] "he is able **to do**"
- @deva[जीवितुं खादति] "he eats **to live**"

The main verb expresses the actual action; the infinitive expresses its goal or purpose.

## @ref[3.4.65] - शकधृषज्ञाग्लाघटरभलभक्रमसहार्हास्त्यर्थेषु तुमुन्

**Key Terms:** @deva[तुमुन्], @deva[शक्], @deva[अर्ह्]

The affix @deva[तुमुन्] is added to a verb when another verb with the following meanings governs it:

| Governing verb | Meaning  | Example                                      |
| -------------- | -------- | -------------------------------------------- |
| @deva[शक्]     | be able  | @deva[कर्तुं शक्नोति] "is able to do"        |
| @deva[धृष्]    | dare     | @deva[वक्तुं धृष्णोति] "dares to say"        |
| @deva[ज्ञा]    | know how | @deva[नेतुं जानाति] "knows how to lead"      |
| @deva[ग्लै]    | be weary | @deva[गन्तुं ग्लायति] "is tired of going"    |
| @deva[रभ्]     | begin    | @deva[पठितुम् आरभते] "begins to read"        |
| @deva[लभ्]     | obtain   | @deva[खादितुं लभते] "gets to eat"            |
| @deva[सह्]     | bear     | @deva[श्रोतुं सहते] "endures to hear"        |
| @deva[अर्ह्]   | deserve  | @deva[ज्ञातुम् अर्हसि] "you deserve to know" |

The @deva[अर्ह्] usage is extremely common in texts: @deva[क्षन्तुम् अर्हसि] "you should forgive" (lit. "you deserve to forgive").

## @ref[7.2.35] - आर्धधातुकस्येड् वलादेः

**Key Terms:** @deva[आर्धधातुक], @deva[इट्]

An @deva[आर्धधातुक] affix beginning with a consonant of @deva[वल्] (most consonants except @deva[य]) gets the augment @deva[इट्] (= @deva[इ]).

Since @deva[तुमुन्] begins with @deva[त्] (a @deva[वल्] consonant) and is @deva[आर्धधातुक], it receives @deva[इट्] for most roots:

| Root        | + @deva[इट्] + @deva[तुम्] | Result                            |
| ----------- | -------------------------- | --------------------------------- |
| @deva[पठ्]  | @deva[पठ् + इ + तुम्]      | @deva[पठितुम्]                    |
| @deva[लिख्] | @deva[लिख् + इ + तुम्]     | @deva[लिखितुम्]                   |
| @deva[भू]   | @deva[भू + इ + तुम्]       | @deva[भवितुम्] (after @deva[गुण]) |
| @deva[नी]   | @deva[नी + इ + तुम्]       | @deva[नेतुम्] (some @deva[अनिट्]) |

But certain roots are @deva[अनिट्] (see next rule) and do **not** take @deva[इट्].

---

## @ref[7.2.10] - एकाच उपदेशेऽनुदात्तात्

**Key Terms:** @deva[इट्], @deva[अनुदात्त], @deva[अनिट्]

The augment @deva[इट्] is **not** added before an affix joined to a root that is monosyllabic and @deva[अनुदात्त] (unaccented) in the @deva[धातुपाठ]. These are @deva[अनिट्] roots.

**@deva[सेट्] vs. @deva[अनिट्] comparison:**

| Type         | Root        | Infinitive       | Has @deva[इट्]? |
| ------------ | ----------- | ---------------- | --------------- |
| @deva[सेट्]  | @deva[पठ्]  | @deva[पठितुम्]   | Yes             |
| @deva[सेट्]  | @deva[भू]   | @deva[भवितुम्]   | Yes             |
| @deva[अनिट्] | @deva[कृ]   | @deva[कर्तुम्]   | No              |
| @deva[अनिट्] | @deva[नी]   | @deva[नेतुम्]    | No              |
| @deva[अनिट्] | @deva[पच्]  | @deva[पक्तुम्]   | No              |
| @deva[अनिट्] | @deva[वच्]  | @deva[वक्तुम्]   | No              |
| @deva[अनिट्] | @deva[दृश्] | @deva[द्रष्टुम्] | No              |

The @deva[सेट्] / @deva[अनिट्] distinction is one of the most important features of the Pāṇinian system for deriving correct forms.

## @ref[7.3.84] - सार्वधातुकार्धधातुकयोः

**Key Terms:** @deva[गुण], @deva[तुमुन्]

@deva[गुण] substitutes for the final @deva[इक्] vowel of a stem before @deva[आर्धधातुक] affixes. Since @deva[तुमुन्] is @deva[आर्धधातुक], @deva[गुण] applies:

- @deva[भू] → @deva[भो] → @deva[भव्] (by @deva[अयादेश]) → @deva[भवितुम्]
- @deva[नी] → @deva[ने] → @deva[नेतुम्]
- @deva[कृ] → @deva[कर्] (by @deva[गुण]) → @deva[कर्तुम्]
- @deva[श्रु] → @deva[श्रो] → @deva[श्रोतुम्]

---

## @ref[6.1.15] - वचिस्वपियजादीनां किति

**Key Terms:** @deva[वच्], @deva[स्वप्], @deva[सम्प्रसारण]

The semi-vowels of @deva[वच्], @deva[स्वप्], and @deva[यजादि] verbs are vocalized (@deva[सम्प्रसारण]) when followed by an affix with indicatory @deva[क्].

Since @deva[तुमुन्] does not have indicatory @deva[क्], @deva[सम्प्रसारण] does not apply here. So @deva[वच्] → @deva[वक्तुम्] (not \*@deva[उक्तुम्]). Compare with @deva[क्त]: @deva[वच्] → @deva[उक्त] (where @deva[क्] triggers @deva[सम्प्रसारण]).

---

## @ref[6.4.66] - घुमास्थागापाजहातिसां हलि

**Key Terms:** @deva[घु], @deva[मा], @deva[स्था]

For @deva[घु] roots (@deva[दा], @deva[धा]) and @deva[मा], @deva[स्था], @deva[गा], @deva[पा], @deva[हा], @deva[सा] — @deva[ई] substitutes for the root vowel before a consonant-initial @deva[आर्धधातुक] affix with indicatory @deva[क्] or @deva[ङ्].

This gives forms like:

- @deva[दा] → @deva[दातुम्] "to give"
- @deva[धा] → @deva[धातुम्] "to place"
- @deva[स्था] → @deva[स्थातुम्] "to stand"
- @deva[पा] → @deva[पातुम्] "to drink" / "to protect"

---

## @ref[7.2.44] - स्वरतिसूतिसूयतिधूञूदितो वा

**Key Terms:** @deva[स्वृ], @deva[सू], @deva[इट्]

A @deva[वलादि] @deva[आर्धधातुक] affix **optionally** takes @deva[इट्] after certain roots: @deva[स्वृ], @deva[सू], @deva[धूञ्], and roots with indicatory long @deva[ऊ].

This creates **optional forms** — both with and without @deva[इट्] are correct:

- @deva[स्वृ]: @deva[स्वर्तुम्] or @deva[स्वरितुम्]

## concept - Infinitives in Sentences

The infinitive appears in several syntactic patterns:

**1. Purpose ("in order to"):**

- @deva[ग्रामं गन्तुं व्रजति] "He goes in order to go to the village"
- @deva[विद्यां ज्ञातुं पठति] "He studies in order to know the knowledge"

**2. Complement of ability/permission:**

- @deva[त्वम् एतत् कर्तुम् अर्हसि] "You ought to do this"
- @deva[न शक्नोमि वक्तुम्] "I am not able to speak"
- @deva[इच्छामि द्रष्टुम्] "I wish to see"

**3. With @deva[अलम्] ("enough, don't"):**

- @deva[अलं शोचितुम्] "Enough of grieving!" (= don't grieve)

**4. With @deva[युक्तम्] ("proper, fitting"):**

- @deva[युक्तं त्वां रक्षितुम्] "It is proper to protect you"

**Common infinitives to memorize:**

| Infinitive       | Root        | Meaning  |
| ---------------- | ----------- | -------- |
| @deva[कर्तुम्]   | @deva[कृ]   | to do    |
| @deva[गन्तुम्]   | @deva[गम्]  | to go    |
| @deva[वक्तुम्]   | @deva[वच्]  | to speak |
| @deva[द्रष्टुम्] | @deva[दृश्] | to see   |
| @deva[श्रोतुम्]  | @deva[श्रु] | to hear  |
| @deva[ज्ञातुम्]  | @deva[ज्ञा] | to know  |
| @deva[दातुम्]    | @deva[दा]   | to give  |
| @deva[स्थातुम्]  | @deva[स्था] | to stand |
| @deva[भवितुम्]   | @deva[भू]   | to be    |
| @deva[जीवितुम्]  | @deva[जीव्] | to live  |
