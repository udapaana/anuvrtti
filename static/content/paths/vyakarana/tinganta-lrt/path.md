## concept - The Simple Future (@deva[लृट्])

@deva[लृट्] is the simple future tense, expressing an action that will happen. Unlike @deva[लट्] (present) and @deva[लङ्] (imperfect) which use @deva[सार्वधातुक] endings, @deva[लृट्] uses the @deva[आर्धधातुक] vikaraṇa @deva[स्य], making it structurally different.

**Core pattern:** root + (@deva[इट्]) + @deva[स्य] + ending

**Paradigm of @deva[भू] (be) — @deva[लृट्] @deva[परस्मैपद]:**

|     | Singular         | Dual             | Plural            |
| --- | ---------------- | ---------------- | ----------------- |
| 3rd | @deva[भविष्यति]  | @deva[भविष्यतः]  | @deva[भविष्यन्ति] |
| 2nd | @deva[भविष्यसि]  | @deva[भविष्यथः]  | @deva[भविष्यथ]    |
| 1st | @deva[भविष्यामि] | @deva[भविष्यावः] | @deva[भविष्यामः]  |

**Paradigm of @deva[कृ] (do) — @deva[लृट्] @deva[परस्मैपद]:**

|     | Singular         | Dual             | Plural            |
| --- | ---------------- | ---------------- | ----------------- |
| 3rd | @deva[करिष्यति]  | @deva[करिष्यतः]  | @deva[करिष्यन्ति] |
| 2nd | @deva[करिष्यसि]  | @deva[करिष्यथः]  | @deva[करिष्यथ]    |
| 1st | @deva[करिष्यामि] | @deva[करिष्यावः] | @deva[करिष्यामः]  |

The @deva[ष्] in @deva[-ष्य-] arises from @deva[स्] → @deva[ष्] by retroflexion after certain vowels and consonants.

**Comparison with @deva[लट्]:**

| Root       | @deva[लट्] (present) | @deva[लृट्] (future) |
| ---------- | -------------------- | -------------------- |
| @deva[भू]  | @deva[भवति]          | @deva[भविष्यति]      |
| @deva[कृ]  | @deva[करोति]         | @deva[करिष्यति]      |
| @deva[गम्] | @deva[गच्छति]        | @deva[गमिष्यति]      |
| @deva[पठ्] | @deva[पठति]          | @deva[पठिष्यति]      |
| @deva[नी]  | @deva[नयति]          | @deva[नेष्यति]       |
| @deva[दा]  | @deva[ददाति]         | @deva[दास्यति]       |
| @deva[वच्] | @deva[वदति]          | @deva[वक्ष्यति]      |

## @ref[3.3.13] - लृट् शेषे च

**Key Terms:** @deva[लृट्], @deva[भविष्यत्]

The affix @deva[लृट्] is used after a verb to express futurity (@deva[भविष्यत्]). The word @deva[शेष] ("remainder") indicates that @deva[लृट्] covers future time not covered by other future tenses (@deva[लुट्]).

**Examples:**

- @deva[करिष्यति] "he will do"
- @deva[गमिष्यति] "he will go"
- @deva[भविष्यति] "it will be"
- @deva[वक्ष्यामि] "I will speak"
- @deva[द्रक्ष्यसि] "you will see"

@deva[लृट्] is the general-purpose future; @deva[लुट्] (periphrastic future) is used for more remote future.

## @ref[3.1.33] - स्यतासी लृलुटोः

**Key Terms:** @deva[स्य], @deva[तासि], @deva[लृट्]

When @deva[लृ] (covering @deva[लृट्] and @deva[लृङ्]) follows, the vikaraṇa @deva[स्य] is added after the root. When @deva[लुट्] follows, @deva[तासि] is added instead.

The @deva[स्य] vikaraṇa is the defining marker of the simple future:

- @deva[कृ] + @deva[स्य] + @deva[ति] → @deva[करिष्यति]
- @deva[भू] + @deva[स्य] + @deva[ति] → @deva[भविष्यति]
- @deva[गम्] + @deva[स्य] + @deva[ति] → @deva[गमिष्यति]

Note: @deva[स्य] is @deva[आर्धधातुक], which triggers @deva[गुण] and potentially @deva[इट्].

## @ref[7.2.35] - आर्धधातुकस्येड् वलादेः

**Key Terms:** @deva[आर्धधातुक], @deva[इट्], @deva[वल्]

An @deva[आर्धधातुक] affix beginning with a @deva[वल्] consonant gets the augment @deva[इट्] (= @deva[इ]).

Since @deva[स्य] begins with @deva[स्] (a @deva[वल्] consonant), @deva[इट्] is inserted for @deva[सेट्] roots:

| Root       | + @deva[इट्]   | + @deva[स्य] | Result          |
| ---------- | -------------- | ------------ | --------------- |
| @deva[भू]  | @deva[भू + इ]  | + @deva[स्य] | @deva[भविष्यति] |
| @deva[गम्] | @deva[गम् + इ] | + @deva[स्य] | @deva[गमिष्यति] |
| @deva[पठ्] | @deva[पठ् + इ] | + @deva[स्य] | @deva[पठिष्यति] |

But @deva[अनिट्] roots do not take @deva[इट्]:

- @deva[कृ] + @deva[स्य] → @deva[करिष्यति] (@deva[इट्] by separate rule @ref[7.2.70])
- @deva[नी] + @deva[स्य] → @deva[नेष्यति] (no @deva[इट्])
- @deva[वच्] + @deva[स्य] → @deva[वक्ष्यति] (no @deva[इट्])

## @ref[7.2.10] - एकाच उपदेशेऽनुदात्तात्

**Key Terms:** @deva[इट्], @deva[अनुदात्त]

The augment @deva[इट्] is **not** added before an affix following a monosyllabic @deva[अनुदात्त] root. This produces the @deva[अनिट्] pattern:

| @deva[अनिट्] root | Future            | @deva[सेट्] root | Future           |
| ----------------- | ----------------- | ---------------- | ---------------- |
| @deva[नी]         | @deva[नेष्यति]    | @deva[भू]        | @deva[भविष्यति]  |
| @deva[दृश्]       | @deva[द्रक्ष्यति] | @deva[पठ्]       | @deva[पठिष्यति]  |
| @deva[वच्]        | @deva[वक्ष्यति]   | @deva[गम्]       | @deva[गमिष्यति]  |
| @deva[पच्]        | @deva[पक्ष्यति]   | @deva[लिख्]      | @deva[लेखिष्यति] |

The @deva[सेट्] / @deva[अनिट्] distinction is crucial for forming correct futures.

## @ref[1.2.4] - सार्वधातुकमपित्

**Key Terms:** @deva[सार्वधातुक], @deva[अपित्], @deva[ङित्]

A @deva[सार्वधातुक] affix not marked with @deva[प्] is treated as @deva[ङित्]. This rule is relevant because the @deva[तिङ्] endings after @deva[स्य] are @deva[सार्वधातुक], and their @deva[ङित्] status affects whether @deva[गुण] applies to them.

## @ref[3.4.77] - लस्य

**Key Terms:** @deva[ल], @deva[आदेश]

In place of @deva[ल] (the abstract @deva[लकार] marker), the actual @deva[तिङ्] endings substitute. For @deva[लृट्], the same @deva[परस्मैपद] and @deva[आत्मनेपद] endings as @deva[लट्] are used, but now they follow @deva[स्य] instead of @deva[शप्]:

- @deva[लट्]: root + @deva[शप्] + @deva[तिप्] → @deva[भवति]
- @deva[लृट्]: root + (@deva[इट्]) + @deva[स्य] + @deva[तिप्] → @deva[भविष्यति]

---

## @ref[8.4.1] - रषाभ्यां नो णः समानपदे

**Key Terms:** @deva[र्], @deva[ष्], @deva[ण]

After @deva[र्] or @deva[ष्], the letter @deva[न्] becomes @deva[ण्] when they occur in the same word. This @deva[णत्व] rule applies in future forms:

- @deva[करिष्यन्ति] — the @deva[र्] in @deva[कर्] causes @deva[ण] in some forms (like @deva[कारणम्])
- @deva[तरिष्यति] — @deva[तर्] + @deva[इष्य] + @deva[ति]
- @deva[हरिष्यति] — @deva[हर्] + @deva[इष्य] + @deva[ति]

---

## @ref[6.4.62] - स्यसिच्सीयुट्तासिषु भावकर्मणोः

**Key Terms:** @deva[स्य], @deva[तासि], @deva[भाव], @deva[कर्मन्]

Before @deva[स्य] and certain other affixes, in @deva[भाव] (impersonal) and @deva[कर्मन्] (passive) voices, special stem changes may occur. This produces passive future forms:

- Active: @deva[करिष्यति] "he will do"
- Passive: @deva[कारिष्यते] "it will be done" (optional @deva[चिण्]-like treatment)

## concept - @deva[लृट्] in Sentences

**Simple future statements:**

- @deva[श्वः गमिष्यामि] "I will go tomorrow"
- @deva[सः आगमिष्यति] "He will come"
- @deva[वयं करिष्यामः] "We will do"

**Questions about the future:**

- @deva[किं करिष्यसि] "What will you do?"
- @deva[कदा आगमिष्यति] "When will he come?"

**Negative future:**

- @deva[न गमिष्यामि] "I will not go"
- @deva[न भविष्यति] "It will not happen"

**Common future forms:**

| Form              | Root        | Meaning    |
| ----------------- | ----------- | ---------- |
| @deva[भविष्यति]   | @deva[भू]   | will be    |
| @deva[करिष्यति]   | @deva[कृ]   | will do    |
| @deva[गमिष्यति]   | @deva[गम्]  | will go    |
| @deva[वक्ष्यति]   | @deva[वच्]  | will speak |
| @deva[द्रक्ष्यति] | @deva[दृश्] | will see   |
| @deva[दास्यति]    | @deva[दा]   | will give  |
| @deva[नेष्यति]    | @deva[नी]   | will lead  |
| @deva[श्रोष्यति]  | @deva[श्रु] | will hear  |
| @deva[ज्ञास्यति]  | @deva[ज्ञा] | will know  |
| @deva[पठिष्यति]   | @deva[पठ्]  | will read  |
