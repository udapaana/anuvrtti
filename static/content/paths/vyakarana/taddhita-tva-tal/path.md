## concept - Abstract Nouns in Sanskrit

The suffixes @deva[त्व] and @deva[तल्] (realized as @deva[ता]) convert any noun or adjective into an abstract noun expressing "the state or nature of being X." They function like English "-ness," "-ity," or "-hood."

**Core pattern:** stem + @deva[त्व] / @deva[ता] → abstract noun (always neuter for @deva[त्व], feminine for @deva[ता])

| Base          | Meaning | + @deva[त्व]       | + @deva[ता]     | Abstract meaning |
| ------------- | ------- | ------------------ | --------------- | ---------------- |
| @deva[गुरु]   | heavy   | @deva[गुरुत्वम्]   | @deva[गुरुता]   | heaviness        |
| @deva[लघु]    | light   | @deva[लघुत्वम्]    | @deva[लघुता]    | lightness        |
| @deva[मनुष्य] | human   | @deva[मनुष्यत्वम्] | @deva[मनुष्यता] | humanity         |
| @deva[देव]    | god     | @deva[देवत्वम्]    | @deva[देवता]    | divinity         |
| @deva[एक]     | one     | @deva[एकत्वम्]     | @deva[एकता]     | oneness, unity   |
| @deva[बहु]    | many    | @deva[बहुत्वम्]    | @deva[बहुता]    | plurality        |
| @deva[सत्य]   | true    | @deva[सत्यत्वम्]   | @deva[सत्यता]   | truth            |

Both suffixes convey the same meaning; the choice is largely stylistic. @deva[त्व] forms (neuter) are more common in philosophical texts, while @deva[ता] forms (feminine) appear frequently in poetry and general usage.

## @ref[5.1.119] - तस्य भावस्त्वतलौ

**Key Terms:** @deva[भाव], @deva[त्व], @deva[तल्]

The affixes @deva[त्व] and @deva[तल्] (= @deva[ता]) come in the sense of "the nature thereof" (@deva[भाव]) after a word in the Genitive case.

The construction: @deva[X-स्य भावः] = @deva[X-त्वम्] or @deva[X-ता]

**Examples:**

- @deva[अश्वस्य भावः] = @deva[अश्वत्वम्] or @deva[अश्वता] "horse-ness" (the state of being a horse)
- @deva[गोः भावः] = @deva[गोत्वम्] or @deva[गोता] "cow-ness"
- @deva[पुरुषस्य भावः] = @deva[पुरुषत्वम्] or @deva[पुरुषता] "manhood"
- @deva[ब्राह्मणस्य भावः] = @deva[ब्राह्मण्यम्] (by a different suffix) or @deva[ब्राह्मणत्वम्] "brahmanhood"

Note: @deva[भाव] here means "abstract state" — the essential nature or condition of being something.

## @ref[6.4.155] - त्वातलोः

**Key Terms:** @deva[त्व], @deva[ता]

Specific stem-final changes occur before @deva[त्व] and @deva[ता]. This rule governs how the base stem adjusts before these suffixes.

**Stem adjustments before @deva[त्व] / @deva[ता]:**

| Stem type      | Base                 | + @deva[त्व]      | + @deva[ता]    |
| -------------- | -------------------- | ----------------- | -------------- |
| @deva[अ]-final | @deva[शुक्ल] (white) | @deva[शुक्लत्वम्] | @deva[शुक्लता] |
| @deva[इ]-final | @deva[शुचि] (pure)   | @deva[शुचित्वम्]  | @deva[शुचिता]  |
| @deva[उ]-final | @deva[गुरु] (heavy)  | @deva[गुरुत्वम्]  | @deva[गुरुता]  |
| @deva[ई]-final | @deva[नदी] (river)   | @deva[नदीत्वम्]   | @deva[नदीता]   |
| Consonant      | @deva[राजन्] (king)  | @deva[राजत्वम्]   | @deva[राजता]   |

For consonant-final stems, the final consonant often drops before @deva[त्व] / @deva[ता] (e.g., @deva[राजन्] → @deva[राज-] before the suffix).

## @ref[7.4.32] - अस्य च्वौ

**Key Terms:** @deva[अ], @deva[त्व], @deva[च्वि]

The @deva[अ] of certain stems undergoes changes in the @deva[च्वि]-construction ("becoming X"). This interacts with @deva[त्व] in philosophical contexts where transformation is discussed.

The @deva[च्वि] construction expresses "becoming what one was not before":

- @deva[कृष्णः अकृष्णः सन् कृष्णी-भवति] "what was not black becomes black"
- @deva[शुक्लः अशुक्लः सन् शुक्ली-भवति]

This relates to @deva[त्व] / @deva[ता] because both address the abstract essence or state.

## @ref[6.1.66] - लोपो व्योर्वलि

**Key Terms:** @deva[लोप], @deva[त्व]

The semi-vowels @deva[व्] and @deva[य्] are elided before @deva[वल्] consonants. This affects the junction between certain stems and @deva[त्व]:

- @deva[भ्रू] (eyebrow) + @deva[त्व] → the @deva[व्] glide does not appear: @deva[भ्रूत्वम्]
- @deva[गो] + @deva[त्व] → @deva[गोत्वम्] (the @deva[व्] that might arise is elided)

## @ref[7.3.84] - सार्वधातुकार्धधातुकयोः

**Key Terms:** @deva[गुण], @deva[त्व]

@deva[गुण] substitutes for @deva[इक्] before @deva[सार्वधातुक] and @deva[आर्धधातुक] affixes. Since @deva[त्व] and @deva[ता] are @deva[तद्धित] (not @deva[आर्धधातुक]), @deva[गुण] does **not** apply before them.

This is why the base vowel is preserved unchanged:

- @deva[गुरु] + @deva[त्व] = @deva[गुरुत्वम्] (not \*@deva[गुरोत्वम्])
- @deva[लघु] + @deva[त्व] = @deva[लघुत्वम्] (not \*@deva[लघोत्वम्])
- @deva[मृदु] + @deva[ता] = @deva[मृदुता] (not \*@deva[मृदोता])

## @ref[5.1.121] - तेः समुदाये

**Key Terms:** @deva[समुदाय], @deva[त्व]

The suffix may express collective sense (@deva[समुदाय]). @deva[बहुत्वम्] = "plurality, the state of being many" — this goes beyond simple abstraction to denote a collective property.

Other collective uses:

- @deva[एकत्वम्] "singularity, oneness"
- @deva[द्वित्वम्] "duality, the state of being two"
- @deva[त्रित्वम्] "triplicity"

These are used extensively in the Aṣṭādhyāyī itself as grammatical technical terms: @deva[एकत्वम्] (singular number), @deva[द्वित्वम्] (dual number), @deva[बहुत्वम्] (plural number).

## concept - @deva[त्व] / @deva[ता] in Philosophical Texts

These suffixes are indispensable in Indian philosophical literature:

| Term               | Meaning                          | Context                         |
| ------------------ | -------------------------------- | ------------------------------- |
| @deva[सत्त्वम्]    | existence, being                 | Vedānta: the essence of reality |
| @deva[जीवत्वम्]    | the state of being a living soul | Vedānta: individual selfhood    |
| @deva[ईश्वरत्वम्]  | lordship                         | theology                        |
| @deva[प्रमाणत्वम्] | validity (of means of knowledge) | Nyāya: epistemology             |
| @deva[अनित्यत्वम्] | impermanence                     | Buddhism: @deva[अनित्यता]       |
| @deva[शून्यता]     | emptiness                        | Mādhyamika philosophy           |
| @deva[एकता]        | unity                            | Advaita: non-duality            |
| @deva[नानात्वम्]   | multiplicity                     | plurality of selves             |

**Reading practice:** Identify the base and meaning:

1. @deva[पशुत्वम्] → @deva[पशु] (animal) → "animal nature"
2. @deva[स्त्रीत्वम्] → @deva[स्त्री] (woman) → "womanhood"
3. @deva[कारणत्वम्] → @deva[कारण] (cause) → "causality"
4. @deva[सुन्दरता] → @deva[सुन्दर] (beautiful) → "beauty"
5. @deva[महत्त्वम्] → @deva[महत्] (great) → "greatness"
