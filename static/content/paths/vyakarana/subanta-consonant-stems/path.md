## concept - @deva[हलन्त] @term[prātipadika]

@deva[हलन्त] (consonant stems) are the most complex declension class. The stem-final @term[vyañjana] interacts with @term[sup] endings through @term[sandhi], assimilation, and special rules. Major sub-types include:

**1. @deva[-अन्] stems (राजन्, आत्मन्):**

| Case      | @deva[राजन्] (king)          | @deva[आत्मन्] (self) |
| --------- | ---------------------------- | -------------------- |
| Nom. Sg.  | @deva[राजा]                  | @deva[आत्मा]         |
| Acc. Sg.  | @deva[राजानम्]               | @deva[आत्मानम्]      |
| Inst. Sg. | @deva[राज्ञा]                | @deva[आत्मना]        |
| Dat. Sg.  | @deva[राज्ञे]                | @deva[आत्मने]        |
| Gen. Sg.  | @deva[राज्ञः]                | @deva[आत्मनः]        |
| Loc. Sg.  | @deva[राज्ञि] / @deva[राजनि] | @deva[आत्मनि]        |
| Nom. Pl.  | @deva[राजानः]                | @deva[आत्मानः]       |
| Gen. Pl.  | @deva[राज्ञाम्]              | @deva[आत्मनाम्]      |

**2. @deva[-अस्] / @deva[-इस्] / @deva[-उस्] stems (मनस्, हविस्, चक्षुस्):**

| Case      | @deva[मनस्] (mind, n.) |
| --------- | ---------------------- |
| Nom. Sg.  | @deva[मनः]             |
| Acc. Sg.  | @deva[मनः]             |
| Inst. Sg. | @deva[मनसा]            |
| Dat. Sg.  | @deva[मनसे]            |
| Nom. Pl.  | @deva[मनांसि]          |

**3. Stop-final stems (@deva[वाच्], @deva[दिश्], @deva[विद्]):**

| Case      | @deva[वाच्] (speech, f.) |
| --------- | ------------------------ |
| Nom. Sg.  | @deva[वाक्]              |
| Acc. Sg.  | @deva[वाचम्]             |
| Inst. Sg. | @deva[वाचा]              |
| Nom. Pl.  | @deva[वाचः]              |

**Key concept:** Consonant stems show **strong/weak alternation**. In strong cases (Nom., Voc., Acc. Sg. of masculine), the stem may lengthen its vowel or add @deva[नुम्]. In weak cases, the bare stem appears.

## @ref[8.2.23] - संयोगान्तस्य लोपः

**Key Terms:** @deva[संयोग], @deva[अन्त], @deva[लोप]

When a word ends in a consonant cluster (@deva[संयोग]), the last consonant is dropped.

This is critical for consonant stems at word-end (Nom. Sg.):

- @deva[भगवन्त्] + @deva[स्] → @deva[भगवान्] (cluster @deva[न्त्स्] → only @deva[न्] remains after simplification)
- @deva[गोमन्त्] + @deva[स्] → @deva[गोमान्]
- @deva[कृतवन्त्] + @deva[स्] → @deva[कृतवान्]

---

## @ref[8.2.30] - चोः कुः

**Key Terms:** @deva[चु], @deva[कु], @deva[पद]

A guttural (@deva[क्], @deva[ख्], @deva[ग्], @deva[घ्]) substitutes for a palatal (@deva[च्], @deva[छ्], @deva[ज्], @deva[झ्]) at word-end or before a @deva[झल्] consonant:

| Stem          | Word-end      | Before vowel  | Explanation                       |
| ------------- | ------------- | ------------- | --------------------------------- |
| @deva[वाच्]   | @deva[वाक्]   | @deva[वाचा]   | @deva[च्] → @deva[क्] at word-end |
| @deva[त्विष्] | @deva[ट्विट्] | @deva[त्विषा] | @deva[ष्] → @deva[ट्] at word-end |
| @deva[दृश्]   | @deva[दृक्]   | @deva[दृशा]   | @deva[श्] → @deva[क्] at word-end |
| @deva[पच्]    | @deva[पक्]    | @deva[पचा]    | @deva[च्] → @deva[क्] at word-end |

Before vowel-initial endings, the original consonant appears unchanged.

---

## @ref[7.1.70] - उगिदचां सर्वनामस्थानेऽधातोः

**Key Terms:** @deva[उगित्], @deva[अञ्च्], @deva[नुम्]

Stems with indicatory @deva[उ] and @deva[अञ्च्]-type stems receive @deva[नुम्] augment in strong cases (@deva[सर्वनामस्थान]):

| Stem            | Weak          | Strong (@deva[नुम्]) | Forms                                       |
| --------------- | ------------- | -------------------- | ------------------------------------------- |
| @deva[भवत्]     | @deva[भवत्]   | @deva[भवन्त्]        | @deva[भवान्] (Nom.), @deva[भवता] (Inst.)    |
| @deva[गोमत्]    | @deva[गोमत्]  | @deva[गोमन्त्]       | @deva[गोमान्] (Nom.), @deva[गोमता] (Inst.)  |
| @deva[प्राञ्च्] | @deva[प्राच्] | @deva[प्राञ्च्]      | @deva[प्राङ्] (Nom.), @deva[प्राचा] (Inst.) |

---

## @ref[6.4.8] - सर्वनामस्थाने चासम्बुद्धौ

**Key Terms:** @deva[सर्वनामस्थान], @deva[दीर्घ]

In a stem ending in @deva[न्], the preceding vowel is lengthened in strong cases except Vocative singular:

- @deva[राजन्] → @deva[राजान्] → @deva[राजा] (Nom. Sg., with @deva[न्]-drop)
- @deva[राजन्] → @deva[राजानम्] (Acc. Sg.)
- @deva[राजन्] → @deva[राजानौ] (Nom./Acc. Du.)
- @deva[राजन्] → @deva[राजानः] (Nom. Pl.)

But Vocative: @deva[हे राजन्] (no lengthening).

This is what produces the characteristic @deva[राजा] / @deva[राजान-] / @deva[राज्ञ-] alternation.

---

## @ref[8.2.7] - नलोपः प्रातिपदिकान्तस्य

**Key Terms:** @deva[न], @deva[लोप], @deva[पद]

The @deva[न्] at the end of a nominal stem is elided when the word is a @deva[पद] (complete word-form):

- @deva[राजन्] → @deva[राजा] (Nom., @deva[न्] drops before @deva[स्])
- @deva[राजन्] + @deva[भ्याम्] → @deva[राजभ्याम्] (@deva[न्] drops before @deva[भ्])
- @deva[राजन्] + @deva[भिः] → @deva[राजभिः]

But @deva[न्] is retained before vowel-initial endings:

- @deva[राजन्] + @deva[ए] → @deva[राज्ञे] (with internal sandhi)
- @deva[राजन्] + @deva[इ] → @deva[राज्ञि]

---

## @ref[8.4.55] - खरि च

**Key Terms:** @deva[झल्], @deva[चर्], @deva[खर्]

Before a voiceless consonant (@deva[खर्]), a voiced stop (@deva[झल्]) is replaced by its voiceless unaspirated counterpart (@deva[चर्]):

- @deva[विद्] + @deva[ता] → @deva[वित्ता] (= @deva[विद् + ता], @deva[द्] → @deva[त्])
- @deva[भेद्] + @deva[ता] → @deva[भेत्ता]
- @deva[युज्] + @deva[ते] → @deva[युक्ते]

This assimilation is a general sandhi rule affecting all consonant-final stems.

---

## @ref[8.2.62] - क्विन्प्रत्ययस्य कुः

**Key Terms:** @deva[क्विन्], @deva[कु]

Stems formed with the @deva[क्विन्] affix show guttural substitution at word-end:

- @deva[वाच्] → @deva[वाक्] (Nom. Sg.)
- @deva[दृश्] → @deva[दृक्] (Nom. Sg.)

---

## @ref[6.1.68] - हलोऽनन्तरायां संयोगः

**Key Terms:** @deva[हल्], @deva[संयोग]

Consonants combine into clusters when no vowel intervenes. This governs how consonant-final stems join with consonant-initial endings, producing the complex internal sandhi of these declensions.

## concept - Summary of Consonant Stem Types

| Type        | Example        | Nom. Sg.      | Inst. Sg.      | Key pattern                  |
| ----------- | -------------- | ------------- | -------------- | ---------------------------- |
| @deva[-अन्] | @deva[राजन्]   | @deva[राजा]   | @deva[राज्ञा]  | strong/weak + @deva[न्]-drop |
| @deva[-मन्] | @deva[आत्मन्]  | @deva[आत्मा]  | @deva[आत्मना]  | strong/weak                  |
| @deva[-वन्] | @deva[मघवन्]   | @deva[मघवा]   | @deva[मघवता]   | strong/weak                  |
| @deva[-अत्] | @deva[भवत्]    | @deva[भवान्]  | @deva[भवता]    | @deva[नुम्] in strong        |
| @deva[-अस्] | @deva[मनस्]    | @deva[मनः]    | @deva[मनसा]    | @deva[स्] → @deva[ः] at end  |
| @deva[-इस्] | @deva[हविस्]   | @deva[हविः]   | @deva[हविषा]   | @deva[ष्]-retroflexion       |
| @deva[-उस्] | @deva[चक्षुस्] | @deva[चक्षुः] | @deva[चक्षुषा] | @deva[ष्]-retroflexion       |
| @deva[-च्]  | @deva[वाच्]    | @deva[वाक्]   | @deva[वाचा]    | palatal → guttural           |
| @deva[-ज्]  | @deva[भिषज्]   | @deva[भिषक्]  | @deva[भिषजा]   | voiced → voiceless           |
| @deva[-श्]  | @deva[दिश्]    | @deva[दिक्]   | @deva[दिशा]    | sibilant → guttural          |

**Reading tip:** When encountering an unfamiliar form, strip the case ending to find the stem, then check if the remaining consonant has undergone word-final simplification (palatals → gutturals, cluster simplification, @deva[न्]-drop).
