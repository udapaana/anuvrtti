## concept - The Absolutive: "Having Done X..."

The absolutive (@deva[क्त्वा]) expresses a completed action before another: "having done X, [he did Y]."

@deva[भोजनं कृत्वा गच्छति] — "Having eaten (lit. having done eating), he goes."

This path traces how Pāṇini's system derives @deva[कृत्वा] from the root @deva[कृ] "to do." Unlike finite verbs, this is a @term[kṛdanta] — a verbal derivative that functions as an indeclinable.

@prakriya[kR + ktvA]

---

## concept - Step 1: The Input

We start with:

1. **Root**: @deva[कृ] "to do, make" (listed as @deva[डुकृञ्] in the Dhātupāṭha)
2. **Meaning**: completed action preceding another action
3. **Target**: the absolutive suffix @deva[क्त्वा]

The @deva[डु] and @deva[ञ्] in @deva[डुकृञ्] are markers (@term[it]) that will be deleted. The actual root is @deva[कृ].

---

## 3.4.21 - Add the Absolutive Suffix

**Key Terms:** ktvā, samānakartṛka

@deva[समानकर्तृकयोः पूर्वकाले] — When actions share the same agent and one precedes the other in time, [the prior action takes @deva[क्त्वा]].

This rule triggers the addition of @deva[क्त्वा]:

> @deva[कृ] + @deva[क्त्वा]

The suffix @deva[क्त्वा] marks the action as completed BEFORE another action by the SAME agent. "Having done" implies "then [the same person] did something else."

---

## 1.3.8 - Identify the Marker

**Key Terms:** laśaku, ataddhita, it

@deva[लशक्वतद्धिते] — In non-@term[taddhita] affixes, initial @deva[ल्], @deva[श्], or gutturals are @term[it].

The @deva[क्] at the beginning of @deva[क्त्वा] is a guttural. Since @deva[क्त्वा] is a @term[kṛt] suffix (not @term[taddhita]), the @deva[क्] is @term[it]:

> @deva[क्त्वा] → @term[it] = @deva[क्], pronounced part = @deva[त्वा]

This @deva[क्] has a crucial function: it makes the suffix @term[kit] ("having @deva[क्] as @term[it]").

---

## 1.1.5 - Block Guṇa

**Key Terms:** kit, ṅit, guṇa

@deva[क्ङिति च] — @term[guṇa] and @term[vṛddhi] do NOT apply when the affix is @term[kit] or @term[ṅit].

Because @deva[क्त्वा] is @term[kit], the root vowel @deva[ऋ] does NOT strengthen to @deva[अर्] (guṇa). Compare:

- @deva[कृ] + @deva[क्त्वा] → @deva[कृत्वा] (no guṇa, kit blocks it)
- @deva[कृ] + @deva[तव्य] → @deva[कर्तव्य] (guṇa applies, no kit)

This is why the @deva[क्] marker exists — to preserve the root vowel.

---

## 1.3.9 - Delete the Marker

**Key Terms:** it, lopa

@deva[तस्य लोपः] — That which is @term[it] undergoes deletion.

The @deva[क्] is now deleted:

> @deva[कृ] + @deva[क्त्वा] → @deva[कृ] + @deva[त्वा]

But the @deva[क्]'s effect (blocking guṇa) has already applied. The marker did its job and disappears.

---

## 6.1.71 - Combine Root and Suffix

**Key Terms:** hrasvasya, piti, kṛti

@deva[ह्रस्वस्य पिति कृति तुक्] — Before a @term[pit] @term[kṛt] suffix, a short vowel gets @deva[तुक्] augment.

Wait — does @deva[क्त्वा] trigger @deva[तुक्]? Let's check: @deva[क्त्वा] is @term[kit] (has @deva[क्]), not @term[pit]. So this rule does NOT apply.

The root @deva[कृ] directly attaches to @deva[त्वा]:

> @deva[कृ] + @deva[त्वा] → @deva[कृत्वा]

---

## 1.1.40 - Mark as Indeclinable

**Key Terms:** ktvā, avyaya

@deva[क्त्वातोसुन्कसुनः] — [Words ending in] @deva[क्त्वा], @deva[तोसुन्], or @deva[कसुन्] are @term[avyaya].

The resulting form @deva[कृत्वा] is now classified as @term[avyaya] — an indeclinable. It takes no case endings. It functions as an adverb modifying the main verb.

---

## concept - The Complete Derivation

| # | Rule | Operation |
|---|------|-----------|
| 1 | @ref[3.4.21] | Add @deva[क्त्वा] (absolutive for prior action) |
| 2 | @ref[1.3.8] | Mark @deva[क्] as @term[it] |
| 3 | @ref[1.1.5] | @term[kit] blocks guṇa |
| 4 | @ref[1.3.9] | Delete @deva[क्] |
| 5 | @ref[1.1.40] | Result is @term[avyaya] |

Final form: **@deva[कृत्वा]** — "having done"

---

## concept - Why the क् Marker Matters

Compare these two suffixes from the same root:

| Suffix | @term[it] status | Root treatment | Result |
|--------|------------------|----------------|--------|
| @deva[क्त्वा] | @term[kit] | No guṇa | @deva[कृत्वा] |
| @deva[तव्य] | Not @term[kit] | Guṇa applies | @deva[कर्तव्य] |

The silent @deva[क्] in @deva[क्त्वा] controls vowel behavior across hundreds of forms. This is the power of the @term[it]-system.

## concept - Try Other Roots

The same pattern applies to other roots:

@prakriya[bhU + ktvA]{collapsed}

@deva[भू] → @deva[भूत्वा] "having become" (no guṇa because @term[kit])

@prakriya[gam + ktvA]{collapsed}

@deva[गम्] → @deva[गत्वा] "having gone" (note: @deva[म्] → zero by special rule)

@prakriya[pac + ktvA]{collapsed}

@deva[पच्] → @deva[पक्त्वा] "having cooked"

## concept - The Absolutive with Prefixes

When a root has a prefix (@term[upasarga]), a different suffix is used: @deva[ल्यप्] instead of @deva[क्त्वा].

@deva[प्र] + @deva[कृ] + absolutive → @deva[प्रकृत्य] (not *@deva[प्रकृत्वा])

This is governed by @ref[7.1.37]: @deva[समासेऽनञ्पूर्वे क्त्वो ल्यप्] — In a compound (except with @deva[अ-नञ्]), @deva[क्त्वा] is replaced by @deva[ल्यप्].

@prakriya[pra + kR + lyap]{collapsed}

## concept - What's Next

Now you've seen how @term[kṛt] suffixes attach differently from @term[tiṅ] endings. Explore:

- **@deva[क्त]** — The past participle suffix (also @term[kit], so no guṇa): @deva[कृ] → @deva[कृत]
- **@deva[तव्य]** — The gerundive (not @term[kit], so guṇa applies): @deva[कृ] → @deva[कर्तव्य]
- **@deva[तृच्]** — The agent noun suffix: @deva[कृ] → @deva[कर्तृ] (with guṇa)

The @term[it]-markers in each suffix determine root treatment.
